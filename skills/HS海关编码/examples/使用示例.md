## 📈 实际应用场景

```
外贸经理：我们有个新产品要出口，先查一下HS编码

输出：完整的HS编码、退税率、监管条件
行动：根据查询结果准备报关资料

```

输入：产品清单和数量

输入：产品和目标国家
输出：HS编码、进口关税、监管要求

```

### 场景4：报关单证审核
```
报关员：审核这批货物的报关单证

## 🎯 最佳实践

### 1. **提供详细信息**
- 提供完整的产品规格
- 上传清晰的产品图片
- 附上技术文档或规格书

### 2. **使用标准术语**
- 使用行业标准产品名称
- 避免使用地方性俚语
- 提供准确的规格参数

### 3. **分批处理**
- 大量产品分批查询
- 复杂产品单独查询
- 重要产品优先查询

### 4. **验证结果**
- 对比多个查询结果
- 检查数据来源可信度
- 必要时进行人工复核

### 5. **保存记录**
- 保存查询结果和输入
- 建立产品归类数据库
- 记录特殊案例和处理方式
